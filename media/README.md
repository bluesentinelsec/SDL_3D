# Media

Shared assets (models, images, audio) used by SDL3D demos and tests.

Each subdirectory contains one asset and must include a `LICENSE.md`
with source URL, author, and license information.

## Assets

| Directory | Description | License |
|-----------|-------------|---------|
| `simple_robot/` | Multi-mesh robot with node hierarchy | CC-BY 4.0 |
| `shiba/` | Textured Shiba dog (4.3k tris) | CC-BY 4.0 |
| `black_dragon/` | Skinned dragon with idle animation (38k tris) | CC-BY-NC 4.0 |
| `bathroom/` | Interior bathroom scene, 106 meshes, 11 PBR materials (62k tris) | CC-BY 4.0 |
