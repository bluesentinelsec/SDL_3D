# Black Dragon

Renders a skinned Black Dragon model with idle animation. Tests
skeleton loading, animation evaluation, skinned mesh drawing, and
multi-material texture mapping.

## Controls

- **Space** — Toggle auto-orbit
- **P** — Pause/resume animation
- **WASD** — Pan camera
- **QE** — Zoom out/in
- **Arrow keys** — Orbit / adjust height
- **Mouse wheel** — Zoom
- **Escape** — Quit
