# Doom Level

First-person walkthrough of a Doom E1M1-inspired level built entirely
from sector definitions — no external model files. Tests the
sector-based level builder for watertight mesh generation. The robot
NPC sprite presentation is authored through the shared sprite manifest
loader rather than hand-wired frame paths in demo code.

6 connected rooms: start room, corridor, nukage room, side passage,
outdoor area, exit room.

## Controls

- **WASD** — Move
- **Mouse** — Look
- **Escape** — Quit
