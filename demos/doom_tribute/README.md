# Doom Tribute

First-person scene that loads a glTF/GLB model and renders it with
SDL3D's PBR lighting. Used as a testbed for model loading, node
hierarchy traversal, and indoor rendering.

Currently loads the Simple Robot model for testing.

## Controls

- **WASD** — Move
- **Mouse** — Look
- **Up/Down arrows** — Fly up/down
- **Left click** — Print camera coordinates
